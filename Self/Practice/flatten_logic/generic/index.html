JavaScript Array Flattening Methods

Method Explanations & Thought Process

1. Recursive Method

This method uses a simple recursion approach: for each element in the array, if it is itself an array, recursively flatten it and concatenate its results; if it's not an array, add it directly to the result. This approach naturally handles any depth of nesting.

2. Iterative Method Using a Stack

Instead of recursion, this method uses a stack data structure to process elements. It repeatedly pops elements from the stack: if an element is an array it pushes its items back to the stack; if not, it adds it to the result. This avoids recursion and can be more performant or safer against call stack size limits.

3. Using Array.prototype.flat()

JavaScript ES2019 introduced the flat() method which can flatten nested arrays to a specified depth (Infinity for full flattening). This is the simplest and most performant method when modern JS environments are available.

4. Reduce with Recursion

This approach uses Array.prototype.reduce() to accumulate values while recursively flattening sub-arrays. It is a functional programming style alternative to the standard recursion method.
