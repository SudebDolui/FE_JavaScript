Methods Explanation & Complexity

This demonstration features multiple algorithms and data structure approaches to flatten nested arrays, following SOLID principles. Each method exhibits a unique strategy with time and space complexity details:

  • Recursive Flatten: Classical recursion traversing nested elements.
  • Stack Flatten: Iterative depth-first search using an explicit stack.
  • Queue Flatten: Iterative breadth-first search using a queue.
  • Memoized Flatten: Dynamic programming inspired; caches results of repeated subarrays.
  • Generator Flatten: Lazy flattening using JavaScript generator functions.
  • Reduce Flatten: Functional programming approach using Array.reduce with recursion.
  • Deque Flatten: Efficient BFS traversal using a custom double-ended queue.
  • Linear Flatten: Linear time, non-recursive solution adapted from StackOverflow.
